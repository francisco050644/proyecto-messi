# ⚽ DataLab - Análisis de Goles de Lionel Messi

## 📌 Descripción del proyecto
Este proyecto hace parte de la asignatura Programación de Computadores 2026-1.  
Consiste en la construcción de una herramienta en Python que permite analizar un dataset real de goles de Lionel Messi.

En esta Entrega 1 (Explorador) se desarrolló un programa en consola que permite cargar, explorar y analizar los datos usando estructuras básicas del lenguaje como bucles, condicionales y funciones.

---

## 📊 Dataset utilizado
- Fuente: Kaggle  
- Nombre: Lionel Messi All Club Goals  
- Archivo utilizado: messi.csv  
- Subconjunto: 50 registros (requerido)

Columnas incluyen:
- Temporada
- Competición
- Club
- Minuto del gol
- Tipo de gol

---

## ❓ Preguntas del proyecto
- ¿En qué competiciones Messi ha marcado más goles?
- ¿En qué minutos del partido se concentran más goles?
- ¿Qué tipo de gol es más frecuente?

---

## ⚙️ Funcionalidades

1. Buscar registros  
2. Estadísticas (máx, mín, promedio)  
3. Filtrar por condición  
4. Ver columnas  

---

## 🛠️ Tecnologías
- Python 3
- open()
- split()

Sin librerías externas.
